Running a simulationÂ¶

The Quick Start provides the simplest way to run simulations. You can run simulations within Microsoft Excel using the Excel Front End or with Windows command and batch files. Both of these methods will run a single simulation locally.

Installing EMOD from the source provides many more options for running simulations. For example, you can run one or more simulations, run simulations locally or on a remote cluster (for large simulations or multiple simulations), or run simulations for debugging the source code.
